# Authentication with google API using Oauth2

# Requirements

- nodejs
- npm
- express
- express-session
- passport
- passport-googleOauth2

# Usage

```cmd
# install dependances

npm i ou npm install

# after install dependencies run app

npm run start
```

go to `http://localhost:3000/` to use application<br><br>
